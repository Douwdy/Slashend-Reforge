Je suis actuellement en formation de développement web chez OpenClassrooms, cependant je code depuis l'âge de 15 ans.
J'ai débuté avec le Java sur Eclipse, puis je me suis intéressé au web par le biais d'un ami, et grâce à ça je me suis initié à l'HTML et au CSS. c'est pour cela que je me suis lancé dans cette formation,
pour me professionnaliser dans un domaine qui me passionne depuis l'adolescence !

React est le premier framework que j'ai appris pour la mise en forme du front-end autrement qu'avec le JavaScript Vanilla.
Au cours de mon apprentissage, j'ai pu apprendre à utiliser les différents composants de React, comme les components, les props, les state, les hooks, les context, les lifecycle, etc.
